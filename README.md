# Random Task Generator

**Автор:** Дулова Ксения

## Описание

«Random Task Generator» — это простое GUI-приложение на Python, которое помогает пользователям случайным образом выбирать задачи из заранее определённых категорий, отслеживать историю и управлять списком заданий.

Программа позволяет:
- Генерировать случайные задачи из категорий «Учёба», «Спорт» или «Работа».
- Добавлять свои собственные задачи с выбором категории.
- Просматривать историю всех сгенерированных задач.
- Фильтровать историю по категориям.
- Автоматически сохранять историю в JSON-файл и загружать её при запуске.

Интерфейс выполнен с использованием стандартной библиотеки `tkinter`.

## Возможности

- Случайный выбор задачи одной кнопкой.
- Категоризация задач (Учёба, Спорт, Работа).
- Защита от добавления пустых строк.
- Сохранение истории между сеансами.
- Фильтр истории по типу задачи.
- История сохраняется в файл `task_history.json`.
